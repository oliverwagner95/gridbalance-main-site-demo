# Grid Balance Main Site Demo

Statische Demo fuer die Hauptseite von Grid Balance.

## Lokal starten

```bash
python3 -m http.server 4177
```

Danach:

```text
http://127.0.0.1:4177/
```

- Problem-/Vergleichslogik aus der Unternehmenspraesentation

- Grid-Control-/Plattformlogik

- Founder-/Vertrauensblock
